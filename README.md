Aliyun serverless(FC) doh proxy

## Get started

### Set FC env

```
UPSTREAM=https://dns.google/dns-query
// or
UPSTREAM=https://dns.nextdns.io/example
```
### Build & upload
```
make build
```

## Use NextDNS
Use nextdns client group. `https://your-fc-url/Mac` or `https://your-fc-url/MyIPhone`
